# Descripicón General
Este repositorio contiene el desarrollo completo del proyecto Análisis de Patrones de Ventas, cuyo objetivo es identificar tendencias, ciclicidades y patrones de comportamiento en ventas por zona, categoría y sector. Incluye un pipeline end-to-end: preparación de datos, EDA, clustering, modelado predictivo, dashboard y recomendaciones accionables.
El proyecto utiliza Python.


# Objetivos del Proyecto
* Detectar tendencias, estacionalidad y ciclos por zona, sector y categoría.
* Optimizar la rotación de inventario y la asignación de recursos.
* Predecir demanda en el corto y mediano plazo.
